### :mega: 프로젝트 소감

<table style="width:500px;">
    <tr>
        <th>한시연</th>
    </tr>
    <tr>
        <td>
          처음으로 자바 콘솔로 구현하는 프로젝트라서 낯설었지만, 프로그램을 처음부터 기획하는 방법을 알 수 있었습니다. 컬렉션 중 ArrayList와 HashMap을 사용해서 더미를 만들고 이를 이용해 구현하는 점에서 어려움이 있습니다. 하지만 이번 프로젝트를 통해서 실력을 향상할 수 있었으며 다른 사람들의 알고리즘을 배울 수 있었습니다. 파일 입출력을 통해서 데이터를 관리할 수 있는 점이 신기했습니다. 파일에 수강, 회원, 관리자 정보에서 데이터 간의 연관성을 생각하여 파일을 업데이트하는 걸 완벽하게 구현을 하지 못했던 게 아쉬웠습니다. 또한, 코드가 효율성과 통일성 있게 프로그래밍하지 못했습니다. 힘들었던 점은 데이터를 콘솔 창에 출력했을 시 요소마다 길이가 다 달라서 깔끔하게 데이터를 출력하기 위해 맞추는 작업에서 시간이 오래 걸렸습니다.
        </td>
    </tr>
